Nicer.md

A beautiful, zero-server markdown reader. Open and read markdown files — beautifully. No server. No backend. Everything runs in your browser.

  • Paste markdown and see it rendered live, with a choice of themes
  • Reader, Write, Split and Source modes — switch as you work
  • Save back to disk: the file you opened is the file you save
  • Native desktop app for macOS, plus a Chrome extension and CLI

Open source under MIT. Source on GitHub.
