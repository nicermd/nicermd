# Tauri

Tauri is a framework for building tiny, fast binaries for all major
desktop and mobile platforms. Developers can integrate any frontend
framework that compiles to HTML, JavaScript, and CSS for building
their user experience while leveraging languages such as Rust, Swift,
and Kotlin for backend logic when needed.

## Get started

Visit the [quickstart guide](https://example.com/start/) to bootstrap
your first Tauri application.
