# Color palette attribution

Nicer.md ships several themes whose color palettes are inspired by widely-recognised open community designs. We use the original palette names (which are commonly treated as the palette's identity in the editor ecosystem) and credit each designer/project, per the spirit of the original licenses.

Each palette here is freely shared under a permissive license (MIT or equivalent) that permits this use with attribution. None of the original code is redistributed; only the colour values and palette name are referenced. CodeMirror's [@codemirror/theme-one-dark](https://github.com/codemirror/theme-one-dark) is the only palette code that ships verbatim, attributed under MIT in [THIRD-PARTY-LICENSES.md](./THIRD-PARTY-LICENSES.md).

The in-product theme picker renders each palette's "Inspired by …" subtitle from the `inspiredBy` field in [`packages/website/src/themes.ts`](./packages/website/src/themes.ts).
